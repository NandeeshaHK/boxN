Dots & Boxes — Static Demo Page

This repository hosts a static page for the Dots & Boxes project. Replace this page with your HTML/JS game, or link to an exported web build.

Play

If you port your game to HTML/JS, place the output in this folder (site/) and deploy — it will be served at https://<your-user>.github.io/<your-repo>/ or a custom domain.

Built for GitHub Pages deployment — see README for deployment steps.
